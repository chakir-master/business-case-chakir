*Tracker*
![alt text](https://github.com/chakir-master/business-case-chakir/image/tracker.png?raw=true)

**Installation**

* Setup mongo (cloud) database
* Install node_modules :
  ```
  npm i
  ```
* Run express server :

```
npm run start:server
```

* Run angular server:
  ```
  ng serve
  ```

 **Go on** `http://localhost:4200` **And that's it !! Enjoy**
